# README

This project is a project to test AI capability to push code

## Hello World - Claude + GitHub Integration Test

Node.js Hello World template untuk test integrasi Claude dengan GitHub.

## Cara pakai

```bash
# Jalankan app
node index.js

# Jalankan test
node test.js
```

## Struktur file

- `index.js` — fungsi utama (`greet`, `add`)
- `test.js` — unit test sederhana tanpa library eksternal
- `package.json` — konfigurasi project
